Your Server, Your Rules

Host It Yourself

PrestonFace 3 was built from the ground up for community hosting. No subscriptions, no always-online DRM. Download the server kit, configure your world, and invite your friends.

Every server owner gets access to the full admin toolkit: spawn rates, loot tables, PvP zones, XP curves, boss scaling, and cosmetic overrides. Run a hardcore permadeath server or a casual loot-fest — it's entirely up to you.

The server binary runs on Windows, Linux, and macOS. Typical hardware: any machine with 4+ cores, 8 GB RAM, and a decent internet connection handles 30 players without breaking a sweat.

# Download & extract the server kit
$ curl -O https://prestonface3.gg/server/pf3-server-latest.tar.gz
$ tar -xzf pf3-server-latest.tar.gz
$ cd pf3-server/

# Configure your world
$ cp config.example.yml config.yml
$ nano config.yml

# Launch
$ ./pf3-server --start
[PF3] Server started on port 7777
[PF3] World "Ashfall Realm" loaded (seed: 48291)
[PF3] Waiting for adventurers...
